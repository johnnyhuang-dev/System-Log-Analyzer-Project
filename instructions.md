
# Instructions

Welcome to the System Log Analyzer.

To use it effectively, please follow these steps:

1. Have a list of system logs, each log in the format [YYYY-MM-DD HH:MM:SS] INFOTYPE: MESSAGE

2. Ensure your logs are in this format, one log oer the other in a list

3. Upload / copypaste your logs into the "uploaded_logs.txt" file

4. Open the "main.py" file

5. Run the program with "python main.py" in the bottom Terminal (or bash shell), 
or press Run button at the top-right corner

6. Follow instructions carefully from the Terminal to see log analysis
